# Panth Advanced Cart Page for Magento 2

Enhance your Magento 2 shopping cart with a free shipping progress bar, quantity +/- buttons, trust badges, order notes, cart savings display, estimated delivery dates, and a branded empty cart page. Works with both Hyva and Luma themes.

## Features

- **Free Shipping Progress Bar** — visual progress toward free shipping threshold
- **Quantity +/- Buttons** — increment/decrement controls on cart items
- **Trust Badges** — configurable security and quality badges (Secure Checkout, Money Back, Free Returns, Fast Shipping, 24/7 Support, Quality Guarantee)
- **Continue Shopping Button** — customizable return-to-shop link
- **Cart Savings Display** — shows total savings from discounts and special prices
- **Estimated Delivery Date** — business-day delivery range calculation
- **Order Notes** — customer order instructions that persist from cart through checkout to admin
- **Enhanced Empty Cart** — branded empty cart page with illustration and trust badges

## Requirements

- PHP 8.1 or higher
- Magento 2.4.4 or higher
- Panth_Core module

## Installation

### Via Composer

```bash
composer require mage2kishan/module-advancedcart
bin/magento module:enable Panth_AdvancedCart
bin/magento setup:upgrade
bin/magento cache:flush
```

## Configuration

Navigate to **Stores > Configuration > Panth Extensions > Advanced Cart Page** to enable features and customize settings.
